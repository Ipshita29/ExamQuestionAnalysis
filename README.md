# Intelligent Exam Question Analysis and Agentic Assessment Design System

This project is an AI-driven educational analytics system that analyzes exam questions and student responses to evaluate question difficulty, identify learning gaps, and support assessment design. The system progresses from ML-based analytics (Milestone 1) to an agentic AI assistant capable of generating structured assessment improvement recommendations (Milestone 2).

## Project Objective
Design and implement an educational analytics system that:
- Analyzes exam question text and student response patterns
- Predicts difficulty or quality levels
- Provides actionable insights for educators
- Extends into an agentic AI assistant for assessment design

## Features
- Text preprocessing and feature extraction (TF-IDF or embeddings)
- Difficulty classification using ML models such as Logistic Regression or Decision Trees
- Student performance pattern analysis
- Basic UI for file upload and results visualization
- Agentic AI assistant for generating structured recommendations
- Optional RAG-based retrieval for improving assessment suggestions
